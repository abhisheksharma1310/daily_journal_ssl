## Web App Installation Instructions

- Clone daily journal project
- Go to the project directory

```
git clone https://github.com/abhisheksharma1310/daily_journal_ssr
cd daily_journal_ssr/
```
- Install all packages 

- Run web app

```
npm install
npm run start
```

- For build package

```
npm run build
```
